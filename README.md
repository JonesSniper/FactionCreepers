## ⚡️ Features

- Throwable Creeper Eggs
- Custom Creeper Eggs
- Normal Creeper
- Rocket Creeper
- Dynamite Creeper
- Sandstorm Creeper
- Sticky Creeper




## 🚀 Installation


After the plugin is compiled, the final jar will be located in the ``/target`` directory. Take the *non-shaded* jar and
upload that into your servers ``/plugins`` folder.
